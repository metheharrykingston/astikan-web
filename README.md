# Astikan Healthcare Trust Landing Page

A responsive React + Vite recreation of the supplied Astikan landing-page UI.

## Stack

- ReactJS + Vite
- Tailwind CSS
- Lucide React icons
- React Icons for social icons
- GSAP hero animation
- AOS scroll animations

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

The interface is built with reusable React components, Tailwind utility classes, SVG artwork, and icon libraries rather than displaying the reference screenshot as the page.
