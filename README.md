# Astikan Healthcare Trust Landing Page

A responsive React + Vite recreation of the supplied Astikan landing-page UI.

## Stack

- ReactJS + Vite
- Tailwind CSS
- Lucide React icons
- React Icons for social icons
- GSAP hero animation
- AOS scroll animations

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

The original supplied screenshot is preserved at `public/astikan-reference.png` for visual comparison. The page itself is built with React components, CSS, SVG, and icon libraries rather than using the screenshot as the interface.
